
===============================================================================
SUMMARY
===============================================================================

A series of blank kart templates to be used with Kartmaker. Made by DragonDePlatino with contributions from other artists!

KC-standard.png		The standard kart template that comes packaged with Kartmaker.
KC-egg.png			A kart with the wheel and seat pushed back to fit plus-sized racers like Eggman.
KC-seatless.png		For truly enormous racers or robotic ones fused directly to the kart.
KC-mini.png			An extra-tiny kart for tiny racers. Or use it as a clown car, I don't know!
KC-crash.png		A template modelled after the standard kart in Crash Team Racing.
KC-blank.png		A meta-template for creating new templates.
palette.gpl			The SRB2 palette in a text format that can be loaded into Aseprite or GIMP.
modelsheet.png		A right and front-facing model sheet that can be imported into Misfit Model 3D.

===============================================================================
ADVICE
===============================================================================

sprites.ase is an aseprite project with the kart templates and the SRB2 palette loaded into it. To export into a format Kartmaker can use, first crop it to the 80x80 area you want. Then go to File -> Export Sprite Sheet and set these options:

Sheet Type:		By Rows
# of Columns:	9
Padding:		None
Trim:			None
Width/Height:	Default
Layers:			Visible layers
Frames:			All frames
Output File:	None
JSON Data:		None
Open sheet:		Yes

Aseprite will open a new tab with the transparent sprite sheet. You can directly paste this into KC-example-blank.png or use it as-is with Kartmaker. If you use one of the 160x160 kart templates, make sure to use the modified properties.txt that comes with them.
